TV Desa Kota Batu

Dalam proyek magang ini, saya membantu menyiapkan dasar backend untuk Portal Berita TV Desa Kota Batu. Saya bertanggung jawab dalam perancangan struktur data, penentuan relasi antar tabel, serta penyusunan alur kerja backend yang akan digunakan untuk mengelola berita dan konten secara terstruktur.

Jobdesk:

  • Menyusun struktur database dan relasi antar data
  • Merancang alur backend untuk input dan pengelolaan berita
  • Menentukan kebutuhan API untuk pengelolaan konten
  • Menyiapkan sistem autentikasi dan otorisasi untuk admin

Fokus Utama:

  • Database rapi dan mudah dikembangkan
  • Alur proses backend yang jelas
  • Struktur sistem yang mudah dikelola
